# Vivaldi-True-Dark-Theme
Vivaldi theme file and CSS mod for proper dark theme with accent colors

## Installation
1. Settings > Themes > Import Theme, install zip file with theme
2. Enable "Allow CSS modifications" in vivaldi://experiments
3. Settings > Appearance > Custom UI Modifications select folder for CSS files
4. Put one CSS file in that folder (TDT_mode_1 or TDT_mode_2)
5. You can change accent color in Settings > Themes > Editor > Colors > Highlight
